# 环境变量配置模板
# 请将以下内容保存为环境变量或复制到系统环境变量设置中

# DashScope API 密钥 (必需)
# 从 https://dashscope.aliyun.com/ 获取您的API密钥
DASHSCOPE_API_KEY=your-dashscope-api-key-here

# 设置方法：
# 1. Windows: 系统属性 -> 环境变量 -> 用户变量 -> 新建
# 2. PowerShell: $env:DASHSCOPE_API_KEY="your-key"
# 3. 命令行: set DASHSCOPE_API_KEY=your-key
# 4. IDE运行配置中设置环境变量
